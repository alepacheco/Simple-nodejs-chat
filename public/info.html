Proyecto de tic

Alejandro Pacheco y Daniel Hidalgo

Objetivo del trabajo

Desarrollar una web dinámica que demostrara la implicación de las ciencias de la información el el campo de las telecomunicaciones

Temetica

La temática es una sala de chat multi-usuario donde el usuario puede experimentar con las tecnologías web

Producto final

La dirección del servidor web es:

  • Chat-Tic

Tareas

  1. Crear la estructura para la pagina de chat
  2. Crear un servidor que maneje los mensajes recibidos y los emita a los cliente
  3. Manejar los mensajes recibidos del servidor (client-side) y mostrarlos en pantalla

Adicional

  • Manejar el control de versiones con Git
  • Configurar CI con TravisCI para prevenir run-time error en producción
  • Configurar un servidor para publicar la ultima versión que haya parado los test de CI automáticamente

Fuentes

  • Stackoverflow
  • js documentación
  • Boostrap documentación

Recursos

  • js para el servidor
  • Boostrap para modelar el chat
  • Git para el control de versiones
  • jQuery para modificar elementos dinámicamente
  • Socket.io para recibir mensajes
  • Express.js para manejar peticiones del servidor
